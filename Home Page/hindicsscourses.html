CSS (कैस्केडिंग स्टाइल शीट्स)

नैनोडिग्री प्रोग्राम

(729)

यह प्रोग्राम छात्रों को CSS का उपयोग करके वेब पेज डिजाइन और स्टाइल करने के कौशल प्रदान करने के लिए लक्षित है, जो उत्तरदायी और दृश्य रूप से आकर्षक लेआउट बनाने पर केंद्रित है। स्नातक वेब विकास और यूजर इंटरफेस डिजाइन भूमिकाओं के लिए तैयार होंगे जिनमें उन्नत CSS विशेषज्ञता की आवश्यकता होती है।

कोर्स में शामिल -

इंटरमीडिएट 40 घंटे वास्तविक दुनिया की परियोजनाएँ कोर्स पूरा होने का प्रमाणपत्र अंतिम अपडेट 30 जून 2024

आप जो कौशल सीखेंगे:

• उत्तरदायी लेआउट बनाएं • CSS ढांचे और पूर्व-प्रोसेसर को मास्टर करें • उन्नत स्टाइलिंग तकनीकों को लागू करें
